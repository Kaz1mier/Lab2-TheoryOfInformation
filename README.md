## 📋 О проекте
Приложение созданное с помощью Java Script, CSS, HTML для шифрования и дешифрования файлого любого формата
на основе сдвиговых регистров с линейной обратной связью (**LFSR**).
---

Данный проект реализует потоковый шифр на основе LFSR с примитивным многочленом:

**P(x) = x^24 + x^4 + x^3 + x + 1**
